# Sensor de movimento

<p align='center'> Implementação de um sensor de movimento com o MPU6050</p>

## Pinos utilizados

Display LCD 240x320:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SCL  -------- (GPIO 18) |
+-------------------------+
| SDA  -------- (GPIO 19) |
+-------------------------+
| RES  -------- (GPIO 20) |
+-------------------------+
| CS   -------- (GPIO 17) |
+-------------------------+
| DC   -------- (GPIO 16) |
+-------------------------+
````

Sensor de movimento MPU6050:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SCL  --------  (GPIO 1) |
+-------------------------+
| SDA  --------  (GPIO 0) |
+-------------------------+
````

Servo motor SG90:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SDA  --------  (GPIO 2) |
+-------------------------+
````
